### Prerequisites

[Python 2.7](https://www.python.org/downloads/)

### Installing

```
git clone https://github.com/itsinprog/disneyShop.git
```

```
cd disneyShop
```

```
pip install -r requirements.txt
```

```
python run.py
```

All products will be stored in lib/data/productStore.json
